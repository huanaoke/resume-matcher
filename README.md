# 智能简历分析与岗位匹配系统

## 项目介绍
基于 Python + Flask 开发的轻量级简历解析与岗位匹配项目。
实现 PDF 简历上传、文本提取、数据处理、匹配评分功能。

## 技术栈
- Python
- Flask
- Pandas
- NumPy
- PyPDF2

## 功能
1. 上传 PDF 简历
2. 自动提取文本内容
3. Pandas 数据结构化
4. NumPy 匹配度评分计算
5. 简易 Web 交互页面

## 运行步骤
1. 安装依赖
pip install -r requirements.txt

2. 启动项目
python app.py

3. 浏览器访问
http://127.0.0.1:5000
